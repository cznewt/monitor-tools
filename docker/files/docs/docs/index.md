# Monitoring

This site is generated from the **deployed monitor-tools configuration** — it always reflects what is actually rendered and applied.

- **Alerts** — every alerting rule loaded into the ruler, grouped by mixin, with severity and runbook links.
- **Runbooks** — the operational runbooks shipped with each mixin.
- **Inventory** — the mixins and dashboards configured per environment.

Regenerate with `render-docs` (run it after `render-all-resources`).
